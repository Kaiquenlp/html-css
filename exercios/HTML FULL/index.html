Formatação de texto

Para iniciar a formatação de texto, existem tags como <b> e <i> para essa tarefa mas elas não possuem significado e focam apenas na forma, sendo assim, são pouco semânticos. Sendo assim, recomendamos que você passe a usar as tags <strong> e <em> para realizar essas mesmas formatações visuais, só que agora com sentido. A tag <strong> significa que o termo delimitado possui força dentro da frase. Logo, ele aparecerá em negrito. Já a tag <em> significa que queremos dar ênfase (do Inglês emphasis) ao termo. Logo, ele aparecerá em itálico. Note que, ao usar <strong> e <em> no lugar de <b> e <i> , damos mais significado aos nossos termos e conteúdos. Como eles vão ser representados visualmente (forma), vai depender das nossas folhas de estilo CSS.

Listas ordenadas, não ordenada e de definições

Listas Ordenadas:

A HTML chama de ordered lists todas aquelas listas onde a ordem dos itens é algo muito importante. Um passo-a-passo para criar um bolo, uma lista de aprovados no vestibular e uma lista com os carros mais caros do mundo são exemplos de listas ordenadas. Para criar uma ordered list, vamos usar a tag <ol> para delimitar a lista e <li> (list item) para identificar cada item da lista
Exemplo de Lista OL :

Carros mais caros do mundo:

  1. Ferrari
  2. Lamborguini
  3. Pagani
  4. Bugatti

Listas Não Ordenadas:

Se você compreendeu a criação de listas ordenadas, com certeza vai entender as unordered lists, também chamadas de listas com marcadores, que são aquelas onde a ordem dos itens não influenciará no significado da lista. Ela é apenas uma ótima maneira para organizar os itens que não apresentam uma classificação necessariamente. Para criar uma unordered list, vamos usar a tag <ul> para delimitar a lista e a tag <li> para criar cada um dos seus itens internos.
Exemplo de Lista Não Ordenada:

Feira do Mês

  • Feijão
  • Arroz
  • Carne
  • Açucar

ID, Class, Pseudo-Class e Pseudo-element.

Os ID'S são utilizados para identidicar itens no html, são muito utilizados para dar uma organização nas CSS e também no Javascript, visto que são usados para identificar areas que serão trabalhadas, as classes assim como os ID's são para a mesma finalidade, porém são mais abrangentes, pois não se pode ter mais de um item com o mesmo ID, logo criamos uma classe para que os itens aninhados dentro daquela mesma classe recebam os mesmos tratamentos, as Pseudo-Clases são utilizadas com : e elas indicam estados de um determinado item, por exemplo, quando utilizamos a psudo-class hover, quando o mouse passa em cima daquele elemento ele muda o estado, podendo mudar cor, tamanho da letra e outras configurações e por fim os Pseudo-elementos, são utilizados para modificar o conteudo periferico dos elementos, por exmplo, se eu quiser que após um link apareça a palavea link sem que eu tenha que colocar no HTML (e assim perder o sentido semantico) eu adiciono um Pseudo-element com :: e coloco por exemplo, ::AFTER ou ::BEFORE, assim aparecerá um novo elemento que tera essa função.
